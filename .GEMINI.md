# .GEMINI.md - aichat

This is a local-first AI chat platform using a multi-container Docker setup. It provides a Textual TUI and a powerful backend with 96 tools accessible via the Model Context Protocol (MCP).

### Key Directories
- `src/aichat/`: Source code for the Textual TUI client.
- `docker/`: Contains the Dockerfiles for all backend microservices.
- `tests/`: Pytest test suite.
- `docker-compose.yml`: The main definition for the entire service stack.
- `Makefile`: Provides convenient targets for managing the project.

### Entrypoints
- **TUI Application:** The `aichat` command, defined in `pyproject.toml`, which executes `src/aichat/cli.py`.
- **Backend Services:** The entire stack is managed by Docker Compose via `docker-compose.yml`.
- **MCP Gateway:** The `aichat-mcp` service, exposed on port `8096`, is the single entry point for all integrated tools.

### Core Commands
Use the `Makefile` for most operations:
- `make build`: Build all service container images.
- `make up`: Start the full application stack in detached mode.
- `make down`: Stop and remove all running containers.
- `make restart`: A full stop and start cycle.
- `make logs`: Follow the logs from all running services.

### Validation Commands
- `make test`: Run the complete pytest suite.
- `make lint`: Run `ruff` linting checks.
- `make smoke`: Perform a quick health check on all service endpoints.
- `make security-checks`: Run a comprehensive set of security analysis tools.
